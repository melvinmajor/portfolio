Microsoft Azure, qu'est-ce que c'est ?

Microsoft Azure est un service de Cloud Computing très performant.
Il permet de profiter de toutes les possibilités des environnements virtuels, avec une puissance de calcul très fortement accru par rapport à ce qui est possible de mettre en place sans avoir de coûts exhorbitants.
De plus, l'environnement est très complet et variés. Il existe plus de 200 produits différents disséminés parmis les principales catégories :

  • Analyse
  • Bases de données
  • Blockchain
  • Bureau virtuel Windows
  • Calcul
  • Conteneurs : système de virtualisation à l'image de Docker (hébergement de plusieur services sur un même serveur physique)
  • DevOps
  • Données multimédias
  • Gestion et gouvernance
  • Hybride : système permettant la mise en place de plusieurs environnements différents reliés à un seul et unique ordinateur par exemple
  • IA + Machine Learning
  • Identité : Azure Active Directory, Information Protection, Domain Services, etc.
  • Intégration : lié notamment aux API, un moyen de communication utilisé à grande échelle pour les développeurs, partenaires et employés en développement informatique
  • Internet des Objets
  • Migration
  • Mise en réseau : DNS, passerelle VPN, protection DDoS, mur pare-feu, etc.
  • Mobile
  • Outils de développement
  • Réalité mixte : Kinect DK (modèles vocaux et vision par ordinateur), Remote Rendering, etc.
  • Sécurité : Azure Sentinel, Centre de sécurité Microsoft 365, etc.
  • Stockage
  • Web : App service, Web Apps, etc.

Les bénéfices du Cloud Computing

Le Cloud Computing permet d'accroître la rentabilité, il est évolutif, élastique, actuel, fiable et sécurisé.

Azure Functions

Il s'agit de code qu'on va rendre disponible en ligne.
Pas besoin d'écrire une API car il est accessible via différentes méthodes : HTTP, Timer, CosmosDB, Blob, Queue, EventGrid, EventHub, ServiceBusQueue, ServiceBusTopic
