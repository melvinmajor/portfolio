Odoo : Le hosting cloud et la gestion d'infrastructure de serveurs

Odoo est un logiciel de gestion d’entreprise.

Une CRM est un logiciel que les commerciaux utilisent dans une entreprise et permet de gérer les opportunités commerciales.
Le CRM de Odoo permet de gérer les « planned activities », créer des offres avec gestion de commentaires par-dessus et signature en ligne, création de site web avec bootstrap, etc.

Odoo est écrit en Python avec une base de données PostgreSQL.
Les opérations les plus rapides avec PostgreSQL sont lorsqu’on sait mettre la totalité de la base de données en mémoire vive.
Avec « Let’s Encrypt », on peut facilement crypter les communications entre un navigateur internet et une plateforme.

Tous les serveurs de Odoo (Odoo SaaS) sont sous Ubuntu LTS.
MUNIN est un plugin pour vérifier l’état du serveur, de la base de données, etc.
Il sert à monitorer.

Avis personnel sur la conférence

Cette conférence était intéressante du point de vue technique.
Odoo n'a pas hésité à nous expliquer plus en détails une partie du fonctionnement interne de leur système et la façon que la vie au sein de l'entreprise est gérée.

Ils sont dans un système d'open-space ne présentant aucune forme d'hiérarchie forte, ce qui permet une cohésion plus importante au sein des équipes de développements.

Cette conférence m'a permis de découvrir un certain intérêt pour le langage de programmation Python.
